Roadmap: WASD + QE keyboard driving

X2 SONIC · design note · pending basis-16 graduation
roadmapteleopreference stream

1 · End state

Drive X2 around a scene with the keyboard — W/A/S/D translation + Q/E turning. SONIC stays a tracker; the keyboard drives the reference stream it tracks.

2 · Design

Key presses → reference-motion stream generator: pick the matching skill from the basis-16 library (start / loop / stop / sideways / turn / in-place turn / idle), stitch clips with yaw rebasing and transition blending, and feed the stream to the tracker closed-loop. The design ports the reference-stream approach from the G1 V6 dual-skill system.

3 · Prerequisite

Basis-16 graduation: all 16 foundation clips at passing cadence. Iter-1000 gate stands at 6/14; the backward family and in-place turn are the main gaps (see the basis-16 report).
